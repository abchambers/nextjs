# Workspace switching

The header workspace badge is the active context for Frontline Forecast.

- **Personal desk** keeps private work separate.
- **Organization workspaces** are for company or school work.
- **Classrooms** are for a specific course and term.
- The platform **Owner** also receives **All workspaces**, an oversight view that does not alter anyone else's privacy settings.

The active selection is saved only in the current browser. New forecast runs store the active organization or classroom ID while remaining private by default. Switching workspaces changes the records shown in Verify; it does not change the selected city or erase an unsent draft.

To grant access later, create an organization/classroom and add an active organization or classroom membership for the user's email/account. The upcoming Control Center People and Classes screens will provide that workflow without direct database work.
