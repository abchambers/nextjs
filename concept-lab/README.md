# Concept Lab

Concept Lab is a static design sandbox for Frontline Forecast. It is deliberately disconnected from the production application.

## Rules

- No imports from `src/app`.
- No shared production CSS or components.
- No authentication, database access, live API calls, analytics, or environment variables.
- No production deployment or replacement of the active product.
- A concept may influence production only through a later, explicit, component-by-component decision.

## Open the first concept

Open `index.html` directly in a browser. It is a standalone visual direction called **Northstar**: a calmer product-family shell that keeps weather, education, and operations visibly separate without changing the current app.

The concept intentionally focuses on hierarchy and product boundaries, not on re-implementing the live application.
